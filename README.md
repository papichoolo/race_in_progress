- Fix root cause of the "car drifts right while braking straight" bug: **every car's `toeDeg` was crab-steering the rear axle**. Indices are FL/FR/RL/RR and `setWheelSteering(i, toe[i])` writes the value verbatim, so `[0, 0, 0.1, 0.1]` meant both rear wheels were permanently steered the same direction — a constant lateral force at the rear contact patch. Real toe-in needs the right-side wheels (idx 1 + 3) negated relative to the left-side (idx 0 + 2) so the pair points *inward*. Mirrored all 7 cars' toe values accordingly (Hatchback / Muscle / Rally `[0, 0, 0.1, -0.1]`; Sport / Supercar `[-0.1, 0.1, 0.2, -0.2]` = front toe-out + rear toe-in race setup; F1 `[-0.1, 0.1, 0.15, -0.15]`; God `[0, 0, 0.05, -0.05]`). Also bumped the lateral-velocity assist: brake-pressed damping 6 → 12 1/s, added a mild 2 1/s coast damp so any residual crab from suspension/load asymmetry decays even off-throttle, fade threshold pushed 10 → 12 m/s. Together: braking straight on slopes no longer pulls sideways, the chassis tracks where it points.
